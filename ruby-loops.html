Ruby Loops & Iterators

Conditionals like most languages, Ruby has Conditional statements: statements that cause code to be executed only if a a condition is met. An expression is evaluated, and if its result is true, the code in the conditionals body is executed. If not, it's skipped.

Let's try an example of a loop by creating a file named loop_codelearn.rb 


 
        # loop_codelearn.rb

        if 1  <  2
            puts "Its true! "
        end


As with most other languages, Ruby supports multiple branches in the condition. These statements take the form if/elsif/else.


    
        if score == 100
            puts "Perfect! "
        elsif score >= 70            
            puts  "you pass"
        else
            puts "Summer school time!"
        end


Conditionals rely on Boolean expression to decide whether the code they contain should be executed. Ruby has contants representing the two Boolean values, ture and false. 


          
        if  true
            puts " I'll be printed"
        end
        

        if  false
            puts "I won't!"
        end


Ruby also has all the comparison operators you are used to .


          
        if 1 == 1
            puts "I'll be printed!"
        end


        if 1  >  2
            puts "I won't!"
        end


        if 1  <  2
            puts "I'll be printed!"
        end


        if 1  >= 2 
            puts "I won't"
        end


        if 2  <= 2 
            puts "I'll be printed!"
        end


        if 2  != 2
            puts "I won't!"
        end


It has the Boolean negation operator, which lets you take a true value and make it false or a false value and make it true. 


          
        if ! true
            puts "I wn't be printed"
        end

        if ! false
            puts "I will"
        end
