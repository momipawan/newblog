Create and Assign Ruby Variables

You don't have to declare variables in Ruby; assigning to them creates them. You assign to a variable with the = symbol

 
          if you type              irb displays
          

          small         =   8            =>  8
          

          medium     =   12         =>  12

A variable name starts with a lowercase letter, and can contain letters, numbers, and underscores.

 
        if you type                 irb displays
        

        # 8  is assigned to small  varibale
        # 12 is assigned to medium variable
        

        small     +      medium       =>  20

Variables don't have types in Ruby, they can hold any value you want. You can assign a string to a variable, then immediately assign a floating-point numbers to the same variable, and its perfect legal. 

 
        if you type               irb displays
    
        pie       =   "Lemon"   =>  "Lemon"

        pie       =    3.14         =>   3.14

The += operator lets you add on to the existing value of a variable

 
      if you type                  irb displays
      

      number   =    3               3  
      

      number   +-   1               4
      

      string   =   "ab"            "ab"
      

      string   +=   "cd"          "abcd"

Use of Strings in Ruby

A string is a series of text characters. You can use things to hold names, email addresses, phone numbers, and a million things . Ruby's strings are special because even very large strings are highly efficient to work with. The easier way to specify a string is to surround is either with double quotos (") or single quotos ('). 

 
          "hello"
          

          'world'
