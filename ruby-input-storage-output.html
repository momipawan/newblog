Ruby Input, storage, and output

Our first requirement is to greet the user by name. To accomplish that, we will need to write a script that gets input from the users, stores that input, and then uses that stored value to create some output. You can copy the below lines of code and run in a ruby terminal.

We can do all this in just a few lines of Ruby code:

 
         # Get My Number Game
         # Written by:  you!
         

         

         # A call to the "puts" method
         puts "welcome to 'Get My Number!'"
         

         

         # A call to the "print" method
         print "What's your name? "
         

         

         # A call to the "gets" method
         # A call to a new variable, "input"
         input  =  gets
         

         


         # Interpolating a value into a string
         puts  "welcome, #{input}"
