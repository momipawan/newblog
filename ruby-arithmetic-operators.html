Ruby's Basic Math Operations and Comparisons

Ruby's basic math Operations work just like they do in most other languages. The (+) symbol is for addition, (-) for subtraction, (*) for multiplication, (/) for division, and (* *) for exponentiation. Example below: 

 
          if you type on terminal

          5.4 - 2.2      =>  3.2 
          

          3 * 4          =>  12 
          

          7 / 3.5        =>  2.0 
          

          2 ** 3         =>  8


You can use (<) and (>) to compare two values and see if one is less than or greater than another. You can also use (==) to see if two values are equal. Example below: 

 
          if you type on terminal

          4 < 6              =>  true
          

          4 > 6              =>  false
          

          2 + 2 == 5      =>  false
