IBM UrbanCode Build Rest Api

The following "POST & DELETE" request methods to add & delete users to a specific team

 
Request

POST   http://{hostname}:{port}/rest2/team/{team}/roleMappings

Here is the example of api using curl method.

curl -k -u "username:password" "http://{hostname}:{port}/rest2/team/{team}/roleMappings" -d '{"user": "$user", "role": "$role"}' -X POST -H "Content-Type: application/json"

        


 
Request
     
DELETE   http://{hostname}:{port}/rest2/team/{team}/roleMappings

Here is the example of api using curl method.

curl -k -u "username:password" "http://{hostname}:{port}/rest2/team/{team}/roleMappings" -d '{"user": "$user", "role": "$role"}' -X DELETE -H "Content-Type: application/json"
