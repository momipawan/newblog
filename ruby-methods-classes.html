Classes, Objects & Methods

In this tutorial, We will create Ruby Classes , Methods and Objects . Method definitions look like this in Ruby:

 
         def print_sum(arg1, arg2)
           print arg1  +  arg2
         end

You will need to add parameters to the method definition. Each argument on the method call gets stored in one of the parameters within the method.

The method body consists of one or more Ruby statements that are executed when the method is called.

Here are two methods for accelerating and sounding th horn. They are about as simple as Ruby methods can be; each methods body has a pair of statements that print strings. 

 
          def accelerate
           puts "Stepping on the gas"
           puts "speeding up"
          end

          def sound_horn
           puts "pressing the horn button"
           puts "Beep beep"
          end

The use_headlights method is only slightly more complex; it takes a single parameters, which is interpolated into one of the output strings 

 
            def use_headlights(brightness)
             puts "Turning on #{brightness}"
             puts "watch out for deer"
            end

calling defined methods in Ruby

You can call method you have defiend just like any other. Lets try out our new vehicle methods

Ruby lets you put calls to your methods anywhere -- even within the same source file where you defined them. Since this is such a simple program at this point. we will just stick the method calls right after the method declarations. 

 
            def accelerate
               puts "stepping on the gas"
               puts "Speeding up"
            end

            def sound_horn
               puts "Pressing the horn button"
               puts "Beep beep!"
            end

            def use_headlights(brightness)
               puts "Turning on #(brightness) headlights"
               puts "Watch out for deer!"
            end

            sound_horn
            accelerate
            use_headlights("high-beam")

When we run the above program from the terminal, we will see the result of our method calls

              Pressing the horn button
              Beep beep!
              Stepping on the gas
              Speeding up
              Turning on high-beam headlights
              Watch out for deer!

How to Create an Class in Ruby

Here's an example of a class we could use : a Dog class.

We use the class keyword to start a new class definition, followed by the name of our new class.

We mark the end of the class definition with the end keyword 

             class Dog
                def talk
                  puts "Bark!"
                end

             def move(destination)
                puts "Running to the #(destination)"
             ends

Creating new instances(objects)

if we call the new method on a class, it will return a new instance of that class. We can then assign that instanc to a variable or do whatever else we need to do with it. 

               fido = Dog.new
               rex  = Dog.new

Once we have one or more instance of the class, we can call their instance methods. We do it in the same way we've callled all other methods on objects so far: we use the dot operator to specify which instance is the method's receiver. 

            fido.talk
            rex.move("food bowl")
